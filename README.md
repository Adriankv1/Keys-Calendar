# Keys Calendar

A collaborative calendar application that allows multiple users to manage their availability and coordinate schedules.

## Features

- User authentication
- Personal calendar management
- Group availability view
- Real-time updates
- Modern, responsive UI

## Tech Stack

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Query
- React Router

### Backend
- Node.js
- Supabase
